=== Create a PVC ===

  kubectl apply -f - <<EOF
  apiVersion: v1
  kind: PersistentVolumeClaim
  metadata:
    name: my-data
    namespace: <your-namespace>
  spec:
    storageClassName: {{ .Values.storageClass.name }}
    accessModes: [ReadWriteOnce]
    resources:
      requests:
        storage: 10Gi
  EOF

=== Use in a Pod ===

  kubectl apply -f - <<EOF
  apiVersion: v1
  kind: Pod
  metadata:
    name: my-app
    namespace: <your-namespace>
  spec:
    containers:
      - name: app
        image: ubuntu:22.04
        command: ["sleep", "infinity"]
        volumeMounts:
          - name: data
            mountPath: /data
    volumes:
      - name: data
        persistentVolumeClaim:
          claimName: my-data
  EOF

=== Requirements ===

{{- if .Values.node.useNbd }}
- NBD (Network Block Device) kernel module: modprobe nbd
- No special kernel version requirements
- Standard nbd-client tool (usually pre-installed)
{{- else }}
- Linux kernel >= 6.0 with CONFIG_BLK_DEV_UBLK=y or =m
- Ubuntu 24.04 (kernel 6.8+) recommended
- ublk_drv kernel module loaded: sudo modprobe ublk_drv
{{- end }}
